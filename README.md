# LostTrackr

LostTrackr — DJ library repair tool for macOS & Windows.

Retrouve et répare les morceaux manquants de tes bibliothèques DJ (Serato).

## Téléchargement

Les installateurs officiels (DMG/PKG macOS, EXE Windows) sont disponibles dans
les [releases GitHub](https://github.com/ShanksThatBoy/losttrackr/releases).
L'application se met ensuite à jour directement depuis l'interface.
