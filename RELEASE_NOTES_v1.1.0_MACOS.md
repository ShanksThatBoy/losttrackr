# LostTrackr v1.1.0 macOS

Version officielle 1.1.0 de LostTrackr pour macOS.

## Nouveautés
- Refonte graphique complète de l’interface V2 : navigation latérale, cartes d’actions, écran de scan, aperçu avant réparation et écran de fin.
- Interface adaptée au plein écran et aux fenêtres réduites, sans faux boutons de fenêtre dans l’UI web.
- Aperçu de réparation avec liste complète déroulable des chemins avant/après et scroll synchronisé.
- Bouton “Ouvrir dans Serato” branché sur macOS pour lancer Serato DJ Pro, Lite ou Serato DJ.
- Notification dédiée si aucun dossier source Serato `_Serato_` n’est trouvé.
- Icônes et assets nettoyés avec fonds transparents.

## Corrections
- Correction du scan qui pouvait classer à tort les fichiers comme ambigus à cause de racines de recherche qui se recoupaient.
- Détection Serato plus stricte pour éviter les faux dossiers temporaires.
- Séparation claire entre résultats fiables, éléments à vérifier et introuvables.

## Distribution
- macOS : DMG et PKG non signés/notarisés pour bêta privée. Gatekeeper peut afficher une alerte.
