# LostTrackr v1.3.5

Analyse des métadonnées dans l'espace DJ Set.

- Nouvelle page `Analyser les métadonnées` : scanne un dossier pour repérer les données manquantes de tes titres (artiste, titre, BPM, clé Camelot, genre).
- Enrichissement via la Base de connaissances LostTrackr, avec repli automatique sur les sources externes (MusicBrainz, etc.).
- Tableau de résultats clair avec statuts : titres identifiés, suggestions probables et non identifiés, plus des statistiques par catégorie.
- Refonte premium de l'interface d'analyse : guidage étape par étape, sélection de dossier repensée et retours d'état visuels.
- Corrections et améliorations diverses (dont le défilement de la page de résultats).
- Système de mise à jour sécurisé et vérifié.
