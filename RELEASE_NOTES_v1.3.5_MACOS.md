# LostTrackr v1.3.5 - macOS

- Nouvelle page `Analyser les métadonnées` dans l'espace DJ Set : scanne un dossier et repère les données manquantes (artiste, titre, BPM, clé Camelot, genre).
- Enrichissement via la Base de connaissances LostTrackr, avec repli automatique sur les sources externes (MusicBrainz, etc.).
- Tableau de résultats avec statuts (identifiés, suggestions probables, non identifiés) et statistiques par catégorie.
- Refonte premium de l'interface d'analyse : guidage, sélection de dossier repensée, retours d'état et défilement corrigé.

Build beta privée non notarise.
