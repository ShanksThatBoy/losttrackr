# LostTrackr v1.2.0 Windows

Version Windows 11 x64 de LostTrackr v1.2.0.

## Nouveautes

- Detection intelligente Serato / rekordbox / Traktor / VirtualDJ.
- Selection du logiciel prefere dans l'ecran de preparation quand plusieurs sources sont detectees.
- Textes UI adaptes au logiciel choisi : crates pour Serato, playlists/collection pour les autres logiciels.
- Moteur Serato conserve pour la reparation complete des chemins perdus.
- Socle d'adaptateurs prepare pour ajouter la reparation rekordbox, VirtualDJ et Traktor.

## Packaging

- Installateur Inno Setup : `LostTrackrSetup-v1.2.0-x64.exe`.
- Installation utilisateur courant ou tous utilisateurs.
- Raccourcis Menu Demarrer et Bureau.

## Notes beta

- Build non signe : SmartScreen peut afficher un avertissement.
- La reparation automatique est active pour Serato dans cette version.
- Les autres logiciels sont detectes et validables, puis seront rendus reparables via adaptateurs dedies.
