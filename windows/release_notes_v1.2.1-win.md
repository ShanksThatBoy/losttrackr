# LostTrackr v1.2.1 Windows

Version Windows 11 x64 de LostTrackr v1.2.1.

## Nouveautes

- Ajout du systeme de mise a jour integre depuis LostTrackr.
- Verification du manifest signe et du SHA256 avant lancement du setup `.exe`.
- Canal de mise a jour beta pret pour `updates.losttrackr.com`.
- Detection intelligente Serato DJ, rekordbox, Traktor et VirtualDJ conservee.

## Notes beta

- Installateur Inno Setup : `LostTrackrSetup-v1.2.1-x64.exe`.
- App non signee : SmartScreen peut afficher une alerte.
