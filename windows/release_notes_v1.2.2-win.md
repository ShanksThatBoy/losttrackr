# LostTrackr v1.2.2 Windows

Version Windows 11 x64 de LostTrackr v1.2.2.

## Nouveautés

- Ouverture maximisée par défaut.
- Page d’accueil ajustée pour mieux afficher le statut du bas.
- Sidebar simplifiée sans badge `V2`; version affichée dans la barre supérieure.

## Installateur

- Installateur Inno Setup : `LostTrackrSetup-v1.2.2-x64.exe`.
- Bêta non signée : SmartScreen peut afficher une alerte.
