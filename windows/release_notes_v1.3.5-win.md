# LostTrackr v1.3.5 - Windows

- Adds the new `Analyze metadata` page in the DJ Set area: scans a folder to detect missing track data (artist, title, BPM, Camelot key, genre).
- Enriches metadata through the LostTrackr Knowledge Base, with automatic fallback to external sources (MusicBrainz, etc.).
- Clear results table with statuses (identified, probable suggestions, unidentified) and per-category statistics.
- Premium redesign of the analysis interface: step-by-step guidance, reworked folder picker, visual state feedback, and fixed results scrolling.
- Builds `LostTrackrSetup-v1.3.5-x64.exe` with Inno Setup.
- This release is unsigned. Windows SmartScreen warnings are expected.
- Microsoft Edge WebView2 Runtime is required for the app UI.
